# 📚 ProfeYa

## Descripción

ProfeYa es una aplicación que conecta alumnos con profesores particulares de manera rápida y sencilla, facilitando la búsqueda de clases de apoyo y consultas académicas.

---

## 🚀 Funcionalidades

* Buscar profesores por materia.
* Contactar docentes fácilmente.
* Coordinar horarios de clases.
* Acceder a apoyo académico personalizado.
* Interfaz simple e intuitiva.

---

## 👨‍🏫 Público objetivo

* Estudiantes que necesiten apoyo escolar o universitario.
* Profesores que deseen ofrecer clases particulares.

---

## 🛠️ Tecnologías utilizadas

* HTML
* CSS
* Git y GitHub
